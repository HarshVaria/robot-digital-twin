# 3D Simulation Module - Robot Digital Twin

## Author
Harsh Varia

## Description
This module contains the 3D visualization components for the Robot Digital Twin project.

## Files
- `components/Simulation/Scene.jsx` - Main 3D scene setup (camera, lights, canvas)
- `components/Simulation/RobotModel.jsx` - 3D robot model
- `components/Simulation/Environment.jsx` - Floor, walls, obstacles
- `components/Sensors/LidarViz.jsx` - LiDAR ray visualization
- `components/PathPlanning/PathVisualizer.jsx` - Path visualization in 3D
- `utils/obstacleData.js` - Obstacle configuration data

## Technologies Used
- React
- Three.js
- @react-three/fiber
- @react-three/drei
